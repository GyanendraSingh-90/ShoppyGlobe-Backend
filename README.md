##  ShoppyGlobe Backend API

This repository contains the backend implementation of the **ShoppyGlobe E-commerce Application**, developed using **Node.js, Express.js, and MongoDB (Atlas)**. The backend provides RESTful APIs for product management, user authentication, and shopping cart operations.

##  Tech Stack

 **Node.js** – JavaScript runtime
 **Express.js** – Web framework
 **MongoDB Atlas** – Cloud database
 **Mongoose** – ODM for MongoDB
 **JWT (JSON Web Token)** – Authentication
 **bcryptjs** – Password hashing
 **Thunder Client** – API testing
